2022 Q4

Our current feature, IP-Intelligence & DNS Monitoring, has just been released in Q4 of the 2022. In Q1, we will focus on bug fixes and improving the overall performance of the tool. We will also be working on adding new threat intelligence feeds to the tool to increase its effectiveness.

2023 Q1

In Q1, we will be adding support for IPv6 to the tool. This will allow the tool to monitor and protect against threats on both IPv4 and IPv6 networks. Additionally, we will be implementing automated IP reputation analysis using Machine Learning algorithms to improve the accuracy of the tool.

2023 Q2

In Q2, we will be integrating eBPFShield with popular SIEM systems for centralized monitoring and alerting. This will make it easier for users to monitor and protect multiple servers from a single location. Additionally, we will be working on adding the ability to detect DNS packets on non-standard ports to improve the coverage and detection capabilities of the tool.

2023 Q3

In Q3, we will be adding a JSON output feature to eBPFShield. This will allow users to easily integrate the tool with a UI dashboard for real-time monitoring and alerting. Additionally, we will be working on improving the usability of the tool by adding more detailed documentation and tutorials.

2023 Q4

In Q4, we will be adding a feature to take action on malicious connections. This will include options such as suspending or killing the process, and taking a core dump of it for forensics. Additionally, we will be working on improving the overall security of the tool by adding encryption and authentication features.

2024 Q1

In 2024 Q1, we will be working on expanding the support for different operating systems. This will include adding support for Windows and macOS, in addition to Linux and Ubuntu. Additionally, we will be working on adding support for non-standard DNS ports to improve the coverage and detection capabilities of the tool.
